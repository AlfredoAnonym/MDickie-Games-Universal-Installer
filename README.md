# MDickie Games Universal Installer
Universal Installer of all MDickie freeware games. It will install all games at once - or, games that you choose to install. It will contain delisted games too, but not all delisted games! Thanks to Mat for allowing me to do this

Files in this repository, will be uploaded gradually. First, whenever I will upload a game - i will upload .exe files first, along with patches, and then rest of the game folders. Don't download anything from it - until I list it here as completed.
